# 기능 요구 사항

- 개요

숫자 야구 게임 구현

- 요구사항

1. 사용자가 서로다른 1부터 9까지 숫자 3개를 입력
2. 컴퓨터의 숫자에 따라 힌트를 출력
    1. 같은 숫자 같은 자리 - 스트라이크
    2. 같은 숫자 다른 자리 - 볼
    3. 다른 숫자 다른 자리 - 낫싱
3. 3개의 숫자를 모두 맞힐 시 게임종료
4. 게임 종료후 게임을 다시 시작하거나 완전히 종료할 수 있다.

- 구현
